## Install on Windows

Windows 10 / 11, 64-bit. macOS and Linux aren't supported right now.

1. Download **`eldritch-client-<version>-windows-amd64.zip`** from the
   [latest release](https://github.com/lukeberry99/the-eldritch-client-releases/releases/latest).
2. Extract it somewhere you can write to, like
   `C:\Users\<you>\Apps\eldritch-client\`.

   > Don't extract it under `C:\Program Files\` - the app updates itself in
   > the background and that folder needs admin rights for every write.
3. Double-click **`eldritch-client.exe`**.

That's it. The app lives in your system tray and keeps an eye on your Star
Citizen log file. Updates download and install themselves quietly.

### First launch

Windows might show a "Windows protected your PC" warning the first time.
Click **More info** then **Run anyway**.

### Uninstall

1. Right-click the tray icon and pick **Quit**.
2. Delete the folder you extracted to.
3. Delete `%LocalAppData%\eldritch-client\` to clear your config and logs.
